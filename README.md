# Projeto: Galeria de imagens

![](./no_images.png)
![](./with_images.png)

Listar ou enviar imagens para o firebase

#### Material:

- React
- Typescript

#### Modo de uso:

1. Abra um terminal ( cmd/propt de comando )
2. Clone o repositório ( git clone https://github.com/RobMota/gallery_images.git )
3. Acesse a pasta ( cd gallery_images )
4. E digite os comandos abaixo:

- `npm install`
- `npm start`

### Modo responsivo:

Deixe o navegador em Responsive Design Mode ( Firefox: Ctrl + Shift + M )
